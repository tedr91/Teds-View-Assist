## Ted's View Assist

A ground-up View Assist frontend: pre-canned Home views (Clock, Handheld, Wall Panel), a consistent bottom navbar, a built-in wallpaper library, and Ted-styled assist overlays. Layers on the View Assist integration; requires Custom Icons (Fluent set) for the navbar.

> Installed **manually** — copy `dashboard/` + `views/` into your View Assist config directory, the same way View Assist's own dashboards/views are distributed (not via HACS). See the [README](https://github.com/tedr91/Teds-View-Assist#readme).
